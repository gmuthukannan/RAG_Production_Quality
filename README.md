# RAG_Production_Quality

Project skeleton created with the following top-level folders:

- data: raw and processed data files
- src: source code
- config: configuration files
- scripts: utility or convenience scripts
- tests: unit / integration tests

Files added:

- `requirements.text` — a list of Python packages (one per line). Install with:

```
python -m pip install -r requirements.text
```

Notes:
- `requirements.text` intentionally uses the `.text` extension per request; rename to `requirements.txt` if you prefer tooling compatibility.
- This is a minimal starter layout. Add module packages under `src` and tests under `tests`.
